# TH3NOMADS Photography Website

A responsive, single-page portfolio website built with plain HTML, CSS and JavaScript. No build tools are required.

## Preview locally
Open `index.html` in a browser, or run a local web server:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Before publishing
1. Replace or refine text in `index.html`.
2. Connect the inquiry form to Formspree, HoneyBook, Pixieset, Studio Ninja or another booking platform.
3. Add your business email, phone and service-area details.
4. Add testimonials only after receiving client permission.
5. Upload to Netlify, Vercel, GitHub Pages or your web host.

## Portfolio images
Optimized copies of the uploaded images are stored under `assets/images`. The original uploaded files are unchanged.
