# TH3NOMADS v2

A responsive GitHub Pages website built with plain HTML, CSS, and JavaScript.

## Publish

Upload the contents of this folder to the root of your GitHub repository:

- `index.html`
- `styles.css`
- `script.js`
- `assets/`
- `galleries/`
- `CNAME`

GitHub Pages should deploy from the `main` branch and root folder.

## Booking form

The inquiry form uses Formspree endpoint:

`https://formspree.io/f/xojgjjrb`

Messages are routed according to your Formspree account settings.

## Videos

When ready, add videos under `assets/videos/` and update the hero or services sections.

## Client galleries

The included gallery is a sample. Static GitHub Pages galleries are public to anyone with the URL. Use Pixieset, Pic-Time, or ShootProof for secure password-protected client delivery.
